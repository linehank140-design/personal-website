# Personal Website — katieagate.com

## About

This is Katie Linehan (Agate)'s personal portfolio website. It showcases her work as an eCommerce and Digital Merchandising Strategist.

## Tech Stack

Static HTML/CSS/JS site hosted on GitHub Pages (custom domain: katieagate.com). No build tools, frameworks, or package managers — just plain files served directly.

## Structure

- `index.html` — Homepage with hero, case studies, articles, and impact metrics
- `about.html` — Bio and hobbies
- `skills.html` — Skills overview
- `resources.html` — Curated resources
- `selected-work.html` — Portfolio of selected work
- `forecasting.html`, `growth.html`, `pet-industry.html`, `platform-admin.html` — Individual case study pages
- `post-builder.html` — Interactive post builder tool
- `contact.html` — Contact page
- `styles.css` — All site styles (single file)
- `content.js` — Shared content data (announcements, articles, etc.)
- `platform.js` — Platform admin page logic
- `post-builder.js` — Post builder page logic

## Deployment

Pushes to `main` are automatically deployed via GitHub Pages. The `CNAME` file maps the custom domain.
